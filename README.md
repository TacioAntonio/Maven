### Installation
1. Front-End
```sh
$ git clone https://github.com/TacioAntonio/Maven
$ cd Maven/backend
$ npm i
$ npm start
```

2. Back-End
```sh
$ git clone https://github.com/TacioAntonio/Maven
$ cd Maven/frontend
$ npm i
$ npm start
```

3. [Http-server](https://www.npmjs.com/package/http-server)
```sh
$ http-server -p 4200 -c-1 dist/<project-name>
```
